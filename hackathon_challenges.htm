Hackathon Challenges

Upshot (document translation)

Sponsor: DCGS-SOF

Leader: Sue Kapr

SOF operators need a mobile app that rapidly identifies the existence of watchlist terms (people, places, events, etc) in foreign language documents. Using only Open Source software, build and/or integrate one or more of the following components:

  • A mobile app that sends photo of documents to a backend service, and displays watchlist hits for each document
  • An API-based OCR service that accepts a photo of a document, and returns the text
  • An API-based language translation service that accepts text, autodetects the language, and returns the results in English
  • An API-based watchlist service that accepts English text, matches text elements to a watchlist, and returns matches (and near-matches, with scoring)

BoxBotix (UAV Sensor)

Sponsor: SOFWERX

Leader: Haris Muratagic

SOF operators need flexible options for mounting sensors to small UAVs. Given a BoxBotix Quadcopter system and three different off-the-shelf sensors (to be provided at the challenge kick-off), design, print, and assemble modular connectors (physical, electrical, and data) to integrate the given sensors to the BoxBotix system.

Rocky (Cyber) "Capture The Flag"

Sponsors: HCC/USF/FC2/MCPA

Leaders: Ken Compres/"Scuba" Steve Gary

SOF operators may need to hack a computer/server using open source software. Using only open source software:

  1. Hack into a server and capture the flag.
  2. Defend the server from other hackers trying to access that server.
  3. Provide a how-to checklist on how you hacked the server and defended it.
  4. Present your hack and defend checklist.

Batman (Locating Objects) Real Coordinates for Location Objects

Sponsor: Wright Brothers Institute

Leader: Bob Lee

SOF Operators need to derive the real coordinates of an object using a cell phone at a distance. Using only Open Source software, build and/or integrate one or more of the following capabilities:

  • Novel interface that provides feedback to the user and reason to have app on the phone
  • Rapid display of derived location on phone (or web-hosted) map
  • Circular probable error of location and proof of accuracy
